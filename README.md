# Flask Starter Project

Simple app to skip the first hour or so of setting up a Flask app.

Includes:

 - Database w/ SQLAlchemy
 - Controllers (Blueprints)
 - Templates
 - Tests
 - CLI w/ Invoke for easily running tests.
